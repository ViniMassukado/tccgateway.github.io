GATEWAY

Gateway” surgiu quando visualizamos um problema evidente vivenciado pelo os alunos todos os dias dentro do colégio, sendo dessa forma onde as oportunidades de emprego eram apresentadas
aos alunos,  dessa maneira ainda utilizando a forma impressa e ficando disponível em apenas um local, dificultando a visualização tanto para a escola que não conseguia ter controle se a
vaga ainda estava disponível, quanto para o aluno que não consegue saber quando aquela vaga foi disponibilizada, tendo somente o acesso presencial no colégio.
O Gateway surgiu para mudar o modo com que os jovens procuram a sua primeira chance no mercado de trabalho, contando com um software com vagas focadas para o público jovem, um modo de avaliação
aonde o empregador e o empregado podem se avaliar, um software pensado para que ambas as partes envolvidas tenham a melhor experiência possível.Contamos com um programa para desktop direcionado as
empresas e escolas, um site direcionado para os alunos conseguirem efetuar seu cadastro no site e posteriormente conseguirem visualizar as vagas que estão disponíveis. Além disso contamos também com um
aplicativo que está em fase de desenvolvimento, onde futuramente unificara as duas plataformas na palma da mão.
